# Automated-Fish-Health-Monitoring-for-Aquaculture-using-Deep-Learning
 Developed a CNN-based model to classify fish as alive or dead using image data. Applied preprocessing and data augmentation for improved model performance. Designed a real-time monitoring system for aquaculture environments.
